# AI‑Managed React Todo App

This project is an **AI-managed** React + TypeScript + Vite application that implements a simple Todo app.  
Documentation and project conventions are maintained so future contributors (human or AI) can quickly understand and extend the codebase.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS (for styling)

## Documentation

All project documentation lives in the `docs/` folder:

- `docs/PRD.md` – Product Requirements Document for the Todo app.
- `docs/ARCHITECTURE.md` – High-level architecture and recommended project structure.

AI and human contributors should keep these documents up to date when making non-trivial changes.

## Cursor Configuration

The root `.cursorrules` file defines best practices for:

- React component structure and state management.
- Tailwind CSS usage and styling conventions.
- Accessibility, UX, and general code quality guidelines.

These rules are intended to guide AI-assisted changes in this repository.

## Getting Started

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Then open the printed local URL in your browser.
